# Set-NexusAnonymousAuth

## SYNOPSIS

Turns Anonymous Authentication on or off in Nexus

## SYNTAX

```powershell
Set-NexusAnonymousAuth [-Enabled] [-Disabled] [<CommonParameters>]
```

## DESCRIPTION

Turns Anonymous Authentication on or off in Nexus

## EXAMPLES

### EXAMPLE 1

```powershell
Set-NexusAnonymousAuth -Enabled
```
