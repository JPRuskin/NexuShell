# Get-NexusScript

## SYNOPSIS

Returns scripts stored for execution in Nexus

## SYNTAX

```powershell
Get-NexusScript [[-Name] <String[]>] [<CommonParameters>]
```

## DESCRIPTION

Returns scripts stored for execution in Nexus

## EXAMPLES

### EXAMPLE 1

```powershell
Get-NexusScript
```

### EXAMPLE 2

```powershell
Get-NexusScript -Name SuperAwesomeScript
```
