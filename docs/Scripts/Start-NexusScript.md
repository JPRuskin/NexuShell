# Start-NexusScript

## SYNOPSIS

Executes a saved script in Nexus

## SYNTAX

```powershell
Start-NexusScript [-Name] <String> [<CommonParameters>]
```

## DESCRIPTION

Executes a saved script in Nexus

## EXAMPLES

### EXAMPLE 1

```powershell
Start-NexusScript -Name TestScript
```
