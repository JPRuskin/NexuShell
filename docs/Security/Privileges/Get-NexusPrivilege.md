# Get-NexusPrivilege

## SYNOPSIS

Retrieve Privilege information from Nexus

## SYNTAX

```powershell
Get-NexusPrivilege [[-PrivilegeId] <String>] [<CommonParameters>]
```

## DESCRIPTION

Retrieve Privilege information from Nexus

## EXAMPLES

### EXAMPLE 1

```powershell
Get-NexusPrivilege
```

### EXAMPLE 2

```powershell
Get-NexusPrivilege -PrivilegeId NuGetProdAdmin
```
